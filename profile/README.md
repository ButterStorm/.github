ButterStorm 不是一个项目，而是一种 **AI 创作范式**：

| 领域 | 我们在做什么 |
|------|--------------|
| 🎵 AI Music | 用 AI 生成 Trap / R&B / 实验电子 / 情绪音乐 |

| 🎨 AI Visual Art | 专辑封面、视觉风格、艺术生成系统 |
| 🔁 AI 创作流程自动化 | 把 LLM、多模态模型接入真实创作流程 |
| 🌐 创作者协作系统 | 让开发者 × 音乐人 × 设计师协同创作 |
